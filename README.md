# 🏃 Khronos Maps

> Agregador de carreras populares en España — mapa interactivo con filtros, datos de múltiples fuentes, y soporte bilingüe.

## Tech Stack

- **Next.js 15** (App Router) + TypeScript (strict)
- **Tailwind CSS v4** — dark theme
- **Framer Motion** — animations
- **Leaflet + OpenStreetMap** — interactive map
- **Drizzle ORM + Turso (libSQL)** — database
- **Cheerio** — web scraping
- **next-intl** — i18n (ES + EN)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
